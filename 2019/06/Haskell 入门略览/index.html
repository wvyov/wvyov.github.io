代数数据类型

从一个新的角度来考虑类型，从类型的值集的大小这一角度，来对类型做出分类。

基本数字(自)

0

不存在值构造器，不能创建出 Void 类型的值。

1

只能构造出一个该类型的值。

2

Bool 或 Maybe Unit 或 Either Unit

只能构造出两个该类型的值。

\(2^{32}\)

Int

只能构造出 \(2^{32}\) 个该字段的值。

如：Pair Bool (Either Bool Unit)

所以 Pair Bool (Either Bool Unit) 的值集大小为 2 * 3 = 6
